Если вы хотите проверить используемую конфигурацию, можете использовать команду 
git config --list

Справка по командам Git
$ git help <команда>
$ git <команда> --help

Для вывода краткой инструкции по использованию использовать опцию -h 
$ git add -h

Можно открыть руководство по команде git config
$ git help config
